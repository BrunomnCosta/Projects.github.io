SecuritEST — API Security Scanner

Scanner de segurança para APIs REST que analisa endpoints, calcula uma pontuação de risco com base no OWASP API Security Top 10 e mantém um histórico de resultados. Implementado e em execução na Azure, com infraestrutura definida como código.

A security scanner for REST APIs that analyses endpoints, scores risk against the OWASP API Security Top 10, and keeps a history of results. Deployed and running on Azure, with infrastructure defined as code.

O problema

The problem

APIs são um dos vetores de ataque mais comuns em aplicações modernas — autenticação fraca, falta de rate limiting, exposição excessiva de dados e configurações de CORS incorretas continuam a aparecer em incidentes reais. Muitas equipas pequenas não têm tempo nem ferramentas para verificar estes pontos de forma sistemática antes de publicar uma API.

O SecuritEST nasceu da vontade de ter uma ferramenta simples: aponta-se a um endpoint, corre-se uma análise alinhada com o OWASP API Security Top 10, e obtém-se uma pontuação de risco e um histórico de resultados ao longo do tempo.

APIs remain one of the most common attack surfaces in modern applications — weak authentication, missing rate limiting, excessive data exposure, and misconfigured CORS keep showing up in real incidents. Many small teams don't have the time or tooling to check for these issues systematically before shipping an API.

SecuritEST was built to be a simple tool for that: point it at an endpoint, run a check aligned with the OWASP API Security Top 10, and get back a risk score plus a history of results over time.

Arquitetura

Architecture

O backend é uma API em FastAPI (Python) que executa as verificações de segurança e calcula a pontuação de risco. O frontend em React permite submeter um endpoint, acompanhar o progresso da análise e consultar o histórico. Os resultados são guardados no Azure Cosmos DB.

A aplicação corre containerizada em Docker, publicada no Azure Container Registry e executada no Azure Container Instances. Toda a infraestrutura — ACR, ACI, Cosmos DB e rede — está definida em Terraform, e o pipeline de CI/CD no GitHub Actions automatiza o build, push da imagem e deploy a cada alteração.

The backend is a FastAPI (Python) service that runs the security checks and computes the risk score. The React frontend lets you submit an endpoint, follow the scan progress, and browse the history. Results are stored in Azure Cosmos DB.

The app runs containerized with Docker, published to Azure Container Registry, and executed on Azure Container Instances. All infrastructure — ACR, ACI, Cosmos DB, and networking — is defined in Terraform, and a GitHub Actions pipeline automates the build, image push, and deploy on every change.
